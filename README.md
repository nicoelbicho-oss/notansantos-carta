# No Tan Santos — Carta digital

Carta digital del restaurante **No Tan Santos — Parrilla | Bar — Club Social**.

Reemplazo de la carta vieja en https://www.restomenuqr.com.ar/notansantos/ por una propia, mobile-first, con la Virgen de Guadalupe como pieza visual central.

## Estado

🚧 **En desarrollo**. Actualmente se publica el mockup HTML estático en GitHub Pages para validar diseño en celular. La versión final va a estar hecha con Astro + Tailwind y desplegada en Vercel.
